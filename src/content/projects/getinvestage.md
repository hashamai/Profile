## OVERVIEW

Getinvestage turns scattered, jargon-heavy market data into clear, instant
answers. Built on a fast React terminal-style dashboard and a Python/FastAPI
backend pulling live data from Finnhub and Yahoo, it puts real-time quotes,
charts, and fundamentals in one clean, deployed workspace.

## PROBLEM

Every retail investor faces the same wall: too much noise, too little insight,
and no time to dig through it. Market data is scattered across sites and
buried in jargon.

## SOLUTION

A RAG-first architecture that goes further than a dashboard: a grounded AI
research assistant that answers real questions with cited sources and live
numbers — not another ChatGPT wrapper, but analysis you can actually trust.

## KEY FEATURES

- Real-time market data — quotes, charts, fundamentals, and watchlists via Finnhub + Yahoo
- Terminal-style dashboard — fast, clean, built for people who read markets
- Grounded AI assistant (RAG-first) — real questions, answers with citations and live figures
- Reliable by design — caching, graceful fallbacks, and handled error states
- Deployed and open — live URL, CI/CD pipeline, clean FastAPI + React architecture
